Remove PINCHZOOMer Instances Example

This example shows how to remove PinchZoomer instances using jQuery. This is useful if you want to remove specific images or all images with zoom capability. To see it in action simply press "Remove Image" button on each image or "Remove All" button. For more info on the code see js/remove_example.js
To use: On a touch device you can use pinch to zoom and drag to move the zoomed image. On a PC you can use mousescroll to zoom and drag to move the zoomed image.
