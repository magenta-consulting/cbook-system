API Example - Gallery with Thumbnails

This example shows a how easy it is to convert PinchZoomer to a product gallery with thumbnails and hotspots using the API. This is perfect for showing products in e-commerce sites whether desktop or mobile. This example also demonstrates the use of mouseover as an alternative option for zooming the content when using desktop. Toggle the switch below to zoom either using mousewheel or mouseover input. Note that it will not affect mobile devices since it will still use pinch gesture to zoom the content.
