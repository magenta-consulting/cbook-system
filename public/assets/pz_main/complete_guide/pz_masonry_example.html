API Example - Masonry Grid with Fullscreen Gallery

In this example, thumbnails using a Masonry grid layout open a fullscreen instance of PinchZoomer. This example also demonstrates how easily the PinchZoomer instance is converted to a gallery by changing its contents and automatically loading images on the fly.
