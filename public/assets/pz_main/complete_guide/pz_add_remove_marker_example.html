API Example - Add or Remove Markers

This example shows how to dynamically add or remove markers. Click the buttons to activate add or remove marker. On the "Add Marker" mode, click/tap anywhere in the content to add a marker. On the "Remove Marker" mode, click/tap any markers to remove them.
Click/Tap any point to add a Marker.
Click/Tap any Marker to remove it.
